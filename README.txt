პროექტი აგებულია ობიექტზე ორიენტირებული პროგრამირების (OOP) პრინციპების გამოყენებით.
ამ მიზნით შექმნილია ორი ძირითადი კლასი:

City — ინახავს ქალაქის შესახებ ინფორმაციას: ქალაქის სახელს, ქვეყანას, განედს (latitude) და გრძედს (longitude).
WeatherRecord — ინახავს ამინდის მონაცემებს: ტემპერატურას, ქარის სიჩქარეს და ამინდის კოდს.

OOP-ის გამოყენებამ შესაძლებელი გახადა ერთმანეთთან დაკავშირებული მონაცემების ერთ ობიექტში გაერთიანება, რის შედეგადაც კოდი გახდა უფრო ორგანიზებული, ადვილად გასაგები და მარტივად გაფართოებადი.
პროექტში გამოყენებული ტექნოლოგიები და კონცეფციები

პროექტის მოკლე აღწერა
Weather Dashboard არის Python-ზე შექმნილი კონსოლური აპლიკაცია, რომელიც Open-Meteo API-ის გამოყენებით მომხმარებელს აწვდის რეალურ დროში ამინდის ინფორმაციას. აპლიკაცია იყენებს OOP მიდგომას (City და WeatherRecord კლასები), ინახავს მონაცემებს JSON ფაილებში, საშუალებას იძლევა შეინახოს საყვარელი ქალაქები, ნახოს ამინდის ისტორია და Folium ბიბლიოთეკის გამოყენებით აჩვენოს შენახული ქალაქები მსოფლიო რუკაზე. 🌍☀️

Main:

პროექტი იყენებს main.py-ს, როგორც აპლიკაციის მთავარ ფაილს, სადაც გაერთიანებულია მენიუ, API ინტეგრაცია და მონაცემების დამუშავების ლოგიკა. მონაცემთა მოდელები გამოყოფილია models.py ფაილში, ხოლო ვალიდაცია ხორციელდება დეკორატორების გამოყენებით.
